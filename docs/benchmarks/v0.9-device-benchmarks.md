## 5. Escalation Policy

- **閾値未達値をそのまま記録・公開する** (新規緩和目標を発明しない、M2-14 と同運用)。
- 未達の起票先: Android RTF → M3-02 (Vulkan) / Godot → M3-11 / Server 75 ms → M3-15。
- **M3-19 四半期 Go/No-go review** (`docs/m3-owner-verification-checklist.md` § 4、暦月 2027-03〜05 頃) に本ページ実測値を判定材料として持ち込む。Kill switch D (Claude Code 以外のコミッター 3 名未満) とは別ゲート。
- **X-06 nightly ダッシュボード** への引き継ぎ: 本 v0.9 実機実測値を X-06 Tier 2 nightly 実機トラッキング (NFR-MT-02) の baseline として登録。
- **watermark / research flag 対象外**: 本ページの計測に用いる weight は Whisper base MIT + piper-plus MIT のみ、CC-BY-NC 系 weight (F5-TTS / Fish-Speech / EnCodec) は使用しない = compliance gate (M2-13) 自動拒否対象、目視確認不要。
