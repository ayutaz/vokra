## (a) What landed (CC-side, this WP)

**`scripts/abi-diff.sh --gate` — the enforcement mode (M5-13-T14a).**

The script previously always exited 0 ("ran to completion regardless of whether a delta was found"). Wiring *that* into a required CI check would have been a **fabricated pass** — a permanently-green empty gate. `--gate` adds real teeth **additively** (the default REPORT mode's exit-0 contract is unchanged, so the historical M3-16-T04 aggregator callers are untouched):

| Delta class | `--gate` verdict | Why |
|-------------|-----------------|-----|
| REMOVED symbol | **exit 1** | breaking — a semver MAJOR bump is required |
| CHANGED symbol (signature) | **exit 1** | breaking — MAJOR bump required |
| ADDED symbol, NOT in the changelog | **exit 1** | undocumented ABI addition (the freeze contract forbids this) |
| ADDED symbol already named in the changelog | pass | additive + recorded = fine (MINOR bump) |
| no delta | pass | clean |

- `--changelog <path>` overrides the file `--gate` greps for an ADDED symbol's name (default `docs/abi-changelog.md`).
- The gate is proven to **actually fail** by 4 negative fixtures in `--self-test` (case 9): (a) an unrecorded ADDED delta → exit 1, (b) the same delta once recorded → exit 0, (c) a REMOVED delta → exit 1 regardless of the changelog, (d) REPORT mode still exits 0 on the same delta (contract regression guard). Synthetic anchor/current/changelog fixtures — the real header and anchors are never touched.
- Zero-dep: pure bash + awk + grep over the existing `classify()` machine stream.

**Verify (M1 iMac, 2026-07-21, run by CC):**
- `bash scripts/abi-diff.sh --self-test` = OK (includes the 4 gate negative cases).
- `bash scripts/abi-diff.sh --gate --anchor v1.0-rc` = OK exit 0 (the current header has NO delta since the rc baseline — the freeze surface is stable).
- `bash scripts/abi-diff.sh --gate --anchor m0` = OK exit 0 (all 18 symbols added since M0 are recorded in `docs/abi-changelog.md`).
- **Adversarial**: `bash scripts/abi-diff.sh --gate --anchor m0 --changelog <empty-file>` = **exit 1** — the gate blocks unrecorded additions. This is the proof it is not an empty gate.
- `bash scripts/gen-c-abi.sh --check` = OK (no C ABI drift — `include/vokra.h` is 33 fn + 11 typedef, unchanged; no new C symbol was introduced by this WP).
