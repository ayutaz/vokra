# Vokra official model-zoo manifest (M5-12-T02 — GA DoD item 2 machine SoT)
#
# One record per zoo model, `key = value` lines separated by a blank line
# (the same mechanism `crates/vokra-eval/src/manifest.rs` parses; the typed
# accessor lives in `crates/vokra-eval/src/zoo.rs`). This file is the input the
# DoD item-2 runner (`vokra_eval::dod`) iterates so "every model degrades < 5 %
# / passes numerical-parity CI" is asserted over a *known set*, not a set that
# can silently shrink.
#
# WHY THIS FILE EXISTS (do not delete a record — mark it excluded):
#   DoD item 2 claims the runner covers "all zoo models". A record that
#   silently vanishes makes that claim vacuously true for the missing model.
#   `scripts/check-zoo-manifest-complete.sh` gates that: every `★ 公式 zoo`
#   (and `⚠ 保留`) row in docs/license-audit.md (the tracked machine SoT) must
#   be claimed here by either a normal record or an `excluded_reason` record.
#   Dropping a model is a code review event, not a diff nobody notices.
#
# FIELDS
#   name           stable id (kebab), also the eval gate key stem
#   family         model family (whisper, kokoro, ...)
#   audit_name     EXACT model-name column (col 2, `**` stripped) of the
#                  matching docs/license-audit.md row — this is how the
#                  completeness gate cross-references catalog <-> manifest.
#                  Whisper's 5 size records all carry the one bundle string.
#   license        weight license (distribution-relevant). Cross-referenced
#                  against docs/license-audit.md; a CC-BY-NC weight here is a
#                  hard error (NFR-LC-04 / X-03).
#   since_version  first Vokra version the model shipped in
#   notes          free text (attribution reminders etc.)
#
# GATED records additionally carry:
#   task           one of: asr tts s2s vad codec  (unknown -> hard error)
#   gguf_env       env var whose presence gates the eval (weight-absent =
#                  skipped_no_weight, never a pass)
#   eval_manifest  path to a hyp/ref Manifest (empty = skipped_no_corpus,
#                  a *distinct* skip from weight-absent). Supplying the corpus
#                  is out of this WP's scope (owner / M5-15 / X-06).
#   quality_metric one of: wer  mel_loss+utmos  roundtrip  parity_only
#                  (must be consistent with task — unknown/mismatched -> error)
#   mos_domain     one of: tts_synthesis  codec_streaming  n_a
#                  (codec_streaming = UTMOS advisory-only, degradation.rs)
#   parity_gate    the numerical-parity gate whose green the owner records as
#                  item-2 evidence (this runner does NOT recompute it)
#
# EXCLUDED records carry instead:
#   excluded_reason  why this ★/⚠ zoo row is not a mel/UTMOS quality-gated
#                    model (KWS / speaker-embedding / enhancement / watermark /
#                    vocoder-component), or is held pending license sign-off.
#                    An excluded record is NOT iterated by the runner, but it
#                    keeps the model visible so "dropped" != "forgotten".
